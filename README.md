# DB 테이블 정보 파일 추출 프로젝트

* DB 테이블 정보를 Excel, CSV, XML, JSON 파일로 추출하는 서비스입니다.

## 개발 환경
* Eclipse 4.23 (2022-03)
* Java 8
* Maven 4.0
* Spring Boot 2.6.7

## 기술 스택
Spring Boot
* Spring Web
* Spring Jdbc
* Spring Boot Devtools
* Spring Boot Configuration Processor

Database
* Tibero
* Oracle
* Mybatis

ETC
* Lombok
* Tomcat Embed Jasper
* Jstl
* Tiles Jsp

Extract File
* Apache Poi
* Apache Poi Ooxml
* Open CSV
* Gson
* Jaxb Api
